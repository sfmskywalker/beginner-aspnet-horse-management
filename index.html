Connect a database with EF Core + SQLite

Goal: Save horses so they persist after you close the app.

We‚Äôll use Entity Framework Core (an ORM) and SQLite (a simple embedded database).

  1. Install the EF Core packages (in Command Prompt):
dotnet add package Microsoft.EntityFrameworkCore.Sqlite
dotnet add package Microsoft.EntityFrameworkCore.Design
  1. (If needed) install the EF CLI:
dotnet tool install --global dotnet-ef
  1. Create Data/AppDbContext.cs:
using HorseManager.Models;
using Microsoft.EntityFrameworkCore;

namespace HorseManager.Data;

public class AppDbContext(DbContextOptions<AppDbContext> options) : DbContext(options)
{
    public DbSet<Horse> Horses => Set<Horse>();
}
  1. Register the database in Program.cs:
using HorseManager.Data; // add at the top
using Microsoft.EntityFrameworkCore; // add at the top

var builder = WebApplication.CreateBuilder(args);

builder.Services.AddRazorPages();
builder.Services.AddServerSideBlazor();

builder.Services.AddDbContext<AppDbContext>(options =>
    options.UseSqlite("Data Source=horses.db"));

var app = builder.Build();
  1. Create the database schema with migrations:
dotnet ef migrations add InitialCreate
dotnet ef database update
Tiny win: You now have a real database file: horses.db.
What‚Äôs a migration?

It‚Äôs a versioned change to your database structure. EF Core generates code to create (or update) tables based on your models. See EF Core Migrations.

Edit & delete horses

Goal: Complete the CRUD.

  1. Add handlers back in Pages/Horses.razor to edit and delete:
async Task EditHorse(Horse h)
{
  // open dialog with a copy, then save into DB if confirmed
  _editing = new Horse { Id = h.Id, Name = h.Name, Age = h.Age, Breed = h.Breed };
  var parameters = new DialogParameters { [nameof(HorseForm.Model)] = _editing };
  var dialog = Dialog.Show<HorseForm>("Edit Horse", parameters);
  var result = await dialog.Result;
  if (!result.Canceled)
  {
    var entity = Db.Horses.Find(_editing.Id);
    if (entity is not null)
    {
      entity.Name = _editing.Name;
      entity.Age = _editing.Age;
      entity.Breed = _editing.Breed;
      Db.SaveChanges();
      _horses = Db.Horses.OrderBy(x => x.Name).ToList();
      StateHasChanged();
    }
  }
}

void DeleteHorse(Horse h)
{
  Db.Horses.Remove(h);
  Db.SaveChanges();
  _horses = Db.Horses.OrderBy(x => x.Name).ToList();
}
Tiny win: You can update and remove horses. That‚Äôs the full CRUD!
